# GreenMeet Flask Web Application

GreenMeet is a web application developed as an academic project using Flask, HTML and CSS.  
The platform allows users to discover events, view event details and register for events through a simple and organized interface.

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja Templates
- Session Management
- Password Hashing

## Project Description

The main objective of this project was to develop a functional web application with both frontend and backend components.  
GreenMeet focuses on event discovery and registration, allowing users to create an account, log in, browse available events, search and filter events, register for events and manage their profile.

The project also includes admin-related features, allowing certain users to access protected areas and manage platform content.

## Main Features

- User registration
- User login and logout
- Password hashing
- Session management
- Event listing
- Event detail pages
- Search and filtering system
- Event registration and cancellation
- User profile area
- Admin-protected dashboard
- Dynamic page rendering using Flask and Jinja templates
